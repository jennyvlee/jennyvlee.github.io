Project 3: Analyzing & Predicting Book Checkout Trends

Project Overview

Project Duration: Jan. 2023 – Mar. 2023

Tools Used: R, ggplot2, Seattle Public Library data

Key Achievements

  • Analyzed book and material checkout trends to optimize inventory.
  • Performed statistical tests (ANOVA, Tukey’s HSD) to identify significant differences between checkout periods.
  • Developed predictive models with an 85% accuracy rate to forecast future checkout volumes.
  • Reduced unnecessary inventory stock by 15% based on trend analysis.
